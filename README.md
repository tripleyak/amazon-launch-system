# Pickleball Launch Command Center — Public Demo

An interactive Amazon product launch intelligence package built around a fictional pickleball brand (**Kestrel**). All brands, people, and specific figures have been anonymized or lightly perturbed; this is a **structural and methodological showcase**, not a real launch plan.

## Live Dashboards

- `public/index.html` — master command center (22 tabs, 19 charts, ~22K lines)
- `public/board-presentation.html` — 14-section board presentation
- `public/financial-forecast.html` — 12-month P&L + 13-week cash flow
- `public/poe-sqp-dashboard.html` — POE / Search Query Performance data explorer
- `public/tracking-dashboard.html` — competitive tracking scorecard

## What This Demonstrates

A complete launch-planning framework across seven domains:

| Domain | Coverage |
|---|---|
| **Command Center** | Investment thesis, hero KPIs, velocity flywheel, scenario projections, TOS risk matrix, acceleration triggers |
| **Market Intelligence** | Competitor scoring, ASIN data, niche POE analysis, seasonality, demand clusters, search-term mining, SQP diagnostics |
| **Advertising & Traffic** | Keyword graduation, 20+ channel traffic grid, TikTok framework, MCF integration, deals hierarchy, 90-day calendar |
| **Product & Listing** | 5-SKU unit economics, COSMO/Rufus knowledge-graph optimization, A+ content, international expansion |
| **Finance** | 12-month P&L (16 OpEx lines), 13-week cash flow, ranking cost model, review-milestone CVR |
| **Tracking & Analytics** | 5-dimension brand scoring, impression-share tracking, convergence matrix, ad-spend bubble chart |
| **Execution** | 90-day timeline with critical-path dependencies, 25-item pre-launch checklist, 19-tool tech stack |

## Tech

Static HTML + vanilla JS + Chart.js. No build step. Deploys to Vercel.

## Note on Data

All brand names, competitor names, founder names, and specific geographic references are **fictional**. Select headline financial figures have been perturbed. This artifact is shared as a **methodology sample** — not actionable market intelligence for any real brand.
